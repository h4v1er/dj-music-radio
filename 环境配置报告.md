```bash

export JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk-25.jdk
```

## 四、待手动安装的组件

### 4.1 Nacos（必选 — 微服务注册/配置中心）

**下载**：选一个能打开的链接
- 官方：https://github.com/alibaba/nacos/releases/download/2.4.3/nacos-server-2.4.3.zip
- 备用：https://nacospub.oss-cn-hangzhou.aliyuncs.com/nacos-server-2.4.3.zip

**安装**：

```bash
# 1. 把下载的 nacos-server-2.4.3.zip 放到 ~/Downloads/
# 2. 运行安装脚本
bash ~/environment/install-nacos.sh
```

**启动**：

```bash
bash ~/environment/nacos/bin/startup.sh -m standalone
# 访问: http://localhost:8848/nacos
# 默认账号密码: nacos/nacos
```

**停止**：

```bash
bash ~/environment/nacos/bin/shutdown.sh
```

---

### 4.2 Elasticsearch（可选 — 音乐全文搜索加分项）

**下载**：https://www.elastic.co/downloads/elasticsearch
（选 macOS ARM 版本）

**安装**：解压到 `~/environment/` 即可

**启动**：

```bash
bash ~/environment/elasticsearch-*/bin/elasticsearch
# 访问: http://localhost:9200
```

安装 `.dmg` 后即可使用 `docker` 命令。
