# 架构设计

本文按当前代码和远端运行环境整理，供答辩 PPT、项目说明书和联调排查使用。

## 1. 架构总览

```text
Vue3 + Vite 前端 5173
  |-- /api/**  -> Gateway 8080
  |-- /music/** -> module-music 8082
  |
  v
Spring Cloud Gateway 8080
  |-- module-chat 8081
  |-- module-music 8082
  |-- module-rec 8083
  |-- module-user 8084

基础设施：
  Nacos 8848       服务注册发现
  MySQL 3306       mall / dj_music_radio
  Redis 6379       推荐热门榜
  RabbitMQ 5672    歌单导入、推荐通知
  Netease API 3000 网易云搜索/播放/歌词/歌单
  DeepSeek API     对话规划回复、歌词情绪分析
  QWeather API     天气查询
```

## 2. 模块职责

| 模块 | 职责 | 关键依赖 |
|:--|:--|:--|
| `gateway` | 统一 HTTP 入口和服务路由 | Nacos、LoadBalancer |
| `module-chat` | AI 对话、WebSocket、天气、时间、定位、调用 music/rec | MySQL、DeepSeek、QWeather、OpenFeign |
| `module-music` | 歌曲、播放器数据、播放队列、歌单、导入、网易云代理、歌词、情绪、收藏、历史 | MySQL、RabbitMQ、Netease API、DeepSeek |
| `module-rec` | 热门榜、行为采集、每日推荐、手动刷新、相似推荐、偏好标签 | MySQL、Redis、RabbitMQ、OpenFeign |
| `module-user` | 注册登录、JWT、用户资料、修改密码、用户入口收藏/历史 | MySQL、Nacos、OpenFeign、module-music |
| `frontend` | 单屏桌面控制台 | Vue3、Vite、Element Plus |

## 3. 关键业务链路

### AI 对话推荐

```text
ChatPanel.vue
  -> WebSocket /chat/ws 或 POST /chat/send
  -> ChatService
  -> DeepSeek 规划工具
  -> music/rec/weather/time/location 工具执行
  -> DeepSeek 基于真实工具结果生成回复
  -> selectedSongs 返回前端
  -> 点击歌曲触发 dj-play-song
  -> MusicPanel 播放
```

### 网易云歌单导入

```text
ImportDialog.vue
  -> /music/netease/playlist?id=...
  -> NeteaseCloudMusicApi:3000
  -> 前端预览歌单
  -> POST /music/playlist/import
  -> RabbitMQ music.playlist.import
  -> PlaylistImportListener
  -> 写 song/playlist/playlist_song
  -> 获取歌词
  -> 提交情绪分析和用户品味更新
```

### 播放队列恢复

```text
MusicPanel.vue
  -> 播放、加入队列、移除队列、切换播放模式
  -> PUT /music/queue/state
  -> user_play_queue / user_player_state
  -> 刷新页面或切换用户
  -> GET /music/queue/state 恢复队列和当前歌曲
```

### 推荐服务

```text
播放/收藏/分享
  -> POST /rec/behavior
  -> user_behavior
  -> Redis ZSET song:hot:rank

每天 02:00
  -> RecommendScheduler
  -> 根据用户 Top 行为歌曲找同流派/协同过滤
  -> emotion_tags / genre / artist / album / duration / 协同过滤
  -> daily_recommend
  -> RabbitMQ rec.exchange 通知
```

## 7. 架构风险

| 风险 | 当前处理 |
|:--|:--|
| Gateway 未统一鉴权 | 当前 user 接口校验 JWT；chat/music/rec 仍按 userId 参数联动 |
| 外部 key 依赖 | DeepSeek/QWeather 改为环境变量，不提交真实 key |
| 网易云版权限制 | 无播放 URL 属于外部限制，前端应允许失败 |
| Java 版本不一致 | 文档要求用 JDK25 脚本构建 |
| `/music` 前端直连 8082 | 开发阶段保留，最终部署可改为统一 `/api/music` |
