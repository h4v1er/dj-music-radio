> 所有队员统一遵守，保证最终产物是一个整体，不是拼凑的

│        │                  │                 │

**禁止**：不要改 `App.vue` 的布局结构（grid 三栏），只改自己负责的组件内部。

## 二、颜色规范

全部定义在 `frontend/src/styles/variables.css`，代码里用 `var(--xxx)` 引用：

```css
/* 背景色 — 用这些，禁止硬编码颜色值 */
var(--color-bg)           /* #0a0a14 页面背景 */
var(--color-surface)      /* #141428 面板背景 */
var(--color-surface-hover) /* 悬停态 */
var(--color-border)        /* #2a2a4a 边框 */

/* 强调色 */
var(--color-primary)      /* #e94560 主色调（红色系）*/
var(--color-accent)       /* #f5a623 辅助色（金色）*/
var(--color-success)      /* #4ecca3 成功/在线 */
var(--color-info)         /* #0f9bff 信息蓝色 */

/* 文字 */
var(--color-text)         /* #e0e0e0 正文 */
var(--color-text-muted)   /* #8888aa 次要文字 */
var(--color-text-dim)     /* #555577 禁用态 */
```

## 三、组件规范

### 面板基础类（已写好，直接复用）

```html
<div class="panel chat-panel">    <!-- .panel 是统一样式 -->
  <div class="panel-title">        <!-- .panel-title 统一标题栏 -->
    <span class="dot"></span>
    面板名称
  </div>
  <!-- 你的内容 -->
</div>
```

### 按钮

```html
<button class="btn-primary">主要操作</button>
<el-button type="primary">Element Plus 按钮</el-button>
```

### 滚动条

已全局统一，不需要额外写 CSS。

## 四、组件对应关系

| 文件 | 负责人 | 改动范围 |
|:-----|:------:|:-----|
| `App.vue` | **不要改** | 布局结构 |
| `ChatPanel.vue` | 队员A | 对话消息、WebSocket、输入 |
| `WeatherWidget.vue` | 队员A | 天气图标、温度、城市 |
| `MusicPanel.vue` | 队员B | 播放器控件、歌单列表 |
| `RecPanel.vue` | 队员C | 榜单、推荐、偏好标签 |
| `UserBar.vue` | 队员D | 登录弹窗、收藏、历史 |
| `variables.css` | **不要改** | 全局颜色变量 |
| `global.css` | **不要改** | 全局样式 |
| `api/index.js` | 各自加 | 在自己模块下新增接口 |

## 五、Element Plus 使用

已全局引入，直接在组件里用：

```html
<el-input v-model="xxx" placeholder="..." />
<el-button type="primary">按钮</el-button>
<el-dialog v-model="visible" title="标题">...</el-dialog>
<el-tag>标签</el-tag>
<el-table :data="list">...</el-table>
```

> 不需要额外 import，直接用。所有组件自动套用 Element Plus 主题。
