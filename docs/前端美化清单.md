## 二、具体做法

### 1. DJ 主题背景动画（队员A）

在 `global.css` 加一个动态粒子或渐变背景：

```css
body::before {
  content: '';
  position: fixed; inset: 0; z-index: -1;
  background:
    radial-gradient(ellipse at 20% 50%, rgba(233,69,96,0.08) 0%, transparent 50%),
    radial-gradient(ellipse at 80% 20%, rgba(15,155,255,0.06) 0%, transparent 50%),
    var(--color-bg);
}
```

效果：页面四角有微弱的红蓝光晕，DJ 氛围感拉满。

---

### 2. 音乐可视化频谱（队员B）

在 `MusicPanel.vue` 的播放器封面下方加一个简易频谱条：

```html
<div class="visualizer">
  <span v-for="i in 20" :key="i"
        class="bar"
        :style="{ height: (15 + Math.random() * 40) + 'px' }" />
</div>
```

```css
.visualizer { display: flex; align-items: flex-end; gap: 2px; height: 50px; }
.bar { width: 3px; background: var(--color-primary); border-radius: 2px;
       transition: height 0.2s; }
```

---

### 3. 卡片悬浮微动效（全员）

每个 `.panel` 加：

```css
.panel { transition: transform 0.2s, box-shadow 0.2s; }
.panel:hover {
  transform: translateY(-2px);
  box-shadow: 0 6px 32px rgba(0,0,0,0.5);
}
```

---

### 4. 过渡动画（全员）

在 `App.vue` 的 `<router-view>` 外包一层（或者不需要，因为单页无路由切换）。组件内部用 CSS `transition` 即可。

---

### 5. 统一 Loading 骨架屏（队员D）

Element Plus 自带 `<el-skeleton>`：

```html
<el-skeleton :loading="loading" animated>
  <template #template>
    <el-skeleton-item variant="text" />
    <el-skeleton-item variant="rect" style="height:200px" />
  </template>
  <template #default>
    <!-- 真实内容 -->
  </template>
</el-skeleton>
```

---

### 6. 播放器封面旋转（队员B）

```css
.album-art.playing {
  animation: spin 8s linear infinite;
  border-radius: 50%;
}
@keyframes spin { to { transform: rotate(360deg); } }
```
