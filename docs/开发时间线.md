## 3. 已完成里程碑

### 里程碑 A：智能对话 DJ

- `POST /chat/send`；
- `/chat/ws` WebSocket；
- `chat_history` 持久化，数据库不可用时内存兜底；
- DeepSeek 工具规划和最终回复；
- 工具白名单：music、rec、weather、time、location；
- 和风天气真实 API；
- 浏览器定位工具；
- 聊天推荐歌曲可点击播放。

### 里程碑 B：音乐中心

- 本地歌曲列表、搜索、详情；
- 网易云搜索、播放 URL、详情、歌词、歌单、封面代理；
- 歌单 CRUD、排序、导入；
- RabbitMQ 异步导入；
- 歌词保存和情绪分析；
- 用户品味画像；
- 收藏和播放历史；
- 前端播放器和音乐子组件。

### 里程碑 C：推荐模块

- Redis ZSET 热榜；
- 用户行为上报和计分；
- 今日推荐；
- 相似歌曲推荐；
- 偏好标签；
- 每日 02:00 定时生成推荐；
- RabbitMQ 推荐通知。

### 里程碑 D：前端整合

- 单屏 DJ 控制台；
- 顶栏时间和天气；
- 左侧 AI 聊天；
- 中间播放器、歌曲、歌单、歌词、情绪；
- 右侧热门榜和推荐；
- 底部用户状态栏；
- 聊天推荐与播放器通过 `dj-play-song` 事件联动。
